Detailed Description

template<typename... TArgs>
class DataFile< TArgs >

Loading/saving array of generic data from/into textfiles. A column of file are determined by template parameters of the class. Can save/load any kind of data as long as operators >> and << exist. TODO: ignore '#' lines when loading
